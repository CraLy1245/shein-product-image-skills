# Output Format And Compliance Checklists

Use this file to format the final answer and run quality control.

## Risk Word Control

Avoid risky or exaggerated text unless explicitly supported and platform-safe.

Avoid:

- Best
- Perfect
- 100% Guarantee
- Premium Quality
- High Quality
- Medical Grade
- Therapeutic
- Cure
- Pain Relief
- Baby
- Infant
- Child / Kids when not clearly intended and safe
- Waterproof when only water-resistant or splash-resistant
- Leakproof unless fully supported
- Non-toxic unless supported by certification
- Eco-friendly unless supported by evidence
- Certified badges that do not exist

Prefer safer wording:

- Water-Resistant Surface
- Easy To Clean
- Stable Support
- Compact Storage
- Easy Access
- Fits Daily Use
- Designed For Organized Storage
- Soft Touch Surface
- Foldable Design
- Space-Saving Storage

## Required Output

Always output:

1. Product Summary
2. Selling Point Extraction
3. Consumer Pain Points
4. Module Selection Report
5. Visual Anchor
6. Image Style Distribution
7. Selected Image Planning Table
8. ImageGen Prompts
9. Execution Plan
10. Final Consistency Checklist

## Image Planning Table Format

For each selected image, include:

- Image Number
- Image Module
- Applied Visual Style
- Visual Anchor Used
- Visual Variation Role
- Purpose
- Purchase Question Answered
- Main Visual Evidence Type
- Main Selling Point
- Layout Direction
- Text Style
- English Copy
- Product Details To Show
- ImageGen Prompt Notes
- Negative Constraints

## Detail Image Text Style Rule

For images after Image 1, text style must be one of:

- small label
- callout label
- dimension label
- corner tag
- before / after label
- short caption

Never use:

- large headline
- big slogan
- poster title
- full-width title bar
- long paragraph
- repeated advertising headline

## Final Checklist

1. Image 1 uses Z-001.
2. Size / Dimension Image is included.
3. SKU image is excluded by default unless explicitly requested.
4. Final image count is adaptive and no more than 11.
5. Every image solves a different purchase question.
6. Product reference is preserved.
7. Visual Anchor is shared across the set.
8. Visual Variation Role is distinct per image.
9. Main Visual Evidence Type is not repeated unnecessarily.
10. Detail images do not use large headlines.
11. Text is short, readable, and English only.
12. Risk words are removed.
13. No unsupported claims or fake badges.
14. ImageGen prompts include negative constraints.
15. ImageGen wait/retry policy is applied when generating.
