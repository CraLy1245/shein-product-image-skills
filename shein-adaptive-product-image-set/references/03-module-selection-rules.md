# Adaptive Module Selection Rules

Use this file to decide image modules and final image count. Do not decide the count first. The maximum is 11 images.

## Mandatory Modules

### Image 1: Z-001 Main Image

Purpose:

- attract clicks
- identify the product quickly
- show the strongest click reason

Layout:

- 1:1 square ratio
- clean modern e-commerce background
- large product hero visual
- product occupies around 55%–70% of the image
- short English headline
- 2–3 concise feature tags
- clean spacing
- strong product visibility
- no excessive icons
- no crowded information

### Size / Dimension Image

Always include one size/dimension image.

Purpose:

- answer size, fit, capacity, compatibility, or scale concerns

Rules:

- use confirmed dimensions only
- use cm and inch when available
- use clean dimension arrows
- include scale reference only when appropriate
- do not invent sizes

## Optional Modules

Choose only when supported by actual product needs.

### Structure Detail Image

Use when product structure affects buying decision.

### Material / Surface Detail Image

Use when material, texture, finish, water-resistant surface, thickness, or cleaning is visible and relevant.

### Core Function Image

Use when the product has a functional mechanism or main operation that needs visual explanation.

### Usage Scenario Image

Use when realistic context improves understanding or purchase desire.

### Multi-Scenario Image

Use when the product is genuinely useful across multiple scenes.

### Capacity / Organization Image

Use for storage, organizer, bag, rack, box, holder, container, or compartment products.

### Installation / Use Steps Image

Use only when setup or use process affects conversion.

### Comparison / Pain-Point Solution Image

Use when the product clearly solves a visible problem and the comparison can be safe and non-exaggerated.

### Trust Detail Image

Use when stitching, fasteners, seams, edges, handles, hinges, support, finish, or other visible quality details reduce hesitation.

## SKU Image Rule

SKU image is excluded by default.

Include SKU image only if the user explicitly asks for SKU display or if SKU differences are critical to purchase and cannot be explained elsewhere.

## Merge / Skip Rule

Merge modules when they answer the same purchase question.

Skip modules that are not supported by product materials.

Every image must have a unique purpose and Main Visual Evidence Type.

## Final Count Rule

The final image count equals the selected modules and must not exceed 11.

Do not add images only to reach a target number.
