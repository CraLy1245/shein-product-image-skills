# Visual System And Style Rules

Use this file to keep the whole SHEIN image set visually unified without making every layout, background, product angle, or callout pattern identical.

## Global Visual Style

All images must use a clean, modern, premium e-commerce style.

- 1:1 square ratio
- English text only
- soft neutral background
- product-focused composition
- large clear product display
- minimal modern typography
- short readable copy
- consistent spacing
- simple callout lines
- rounded detail boxes when needed
- soft studio lighting
- natural product shadow
- no messy background
- no excessive icons
- no fake badges
- no exaggerated claims

## JiaZ E-Commerce Image Style

The image style should be product-first, clean, practical, modern, and suitable for SHEIN / cross-border e-commerce.

Avoid over-luxury decoration, crowded layouts, excessive badges, and long text.

## Visual Anchor

Before writing prompts, define one shared Visual Anchor for the full image set.

The Visual Anchor includes:

- Background family
- Lighting
- Product rendering
- Typography
- Label style
- Accent color
- Overall mood

Every image must inherit the same Visual Anchor.

The Visual Anchor keeps the image set unified, but it must not force the same exact background.

## Visual Variation

Each image must have a distinct Visual Variation Role.

Variation can come from:

- camera angle
- crop
- detail focus
- module layout
- background role within the same family
- scene context
- prop intensity
- callout placement
- product placement
- Main Visual Evidence Type

The set must answer different purchase questions.

Do not repeat the same exact background, same product angle, same callout placement, or same visual task across the full set.

## Applied Visual Styles

Use styles according to module need:

- Z-001 Main Image style
- structure detail style
- material close-up style
- size diagram style
- usage scenario style
- capacity / organization style
- comparison style
- trust detail style

Image 1 establishes the click-oriented visual identity. Later images support conversion and trust without becoming repeated hero images.

## Text Style

Image 1 may use one short headline and 2–3 short feature tags.

Detail images should use small labels, callouts, dimension labels, corner tags, before/after labels, or short captions.

Do not use large headline-style titles on detail images.

## Product Consistency

Use uploaded product images as exact product references.

Preserve:

- shape
- proportions
- color
- material
- texture
- structure
- visible details
- SKU differences

Do not redesign the product or add unsupported accessories.
