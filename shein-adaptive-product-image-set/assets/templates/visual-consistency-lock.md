## Fixed Elements Across The Full Set

- Same background family, not the same exact background
- Same lighting direction and softness
- Same product rendering realism
- Same typography system
- Same label language and line weight, not the same label placement or repeated detail-layout pattern
- Same restrained accent color logic
- Same product-first hierarchy
- Same visual cleanliness

## Required Variation

- Camera angle
- Crop
- Detail focus
- Module layout
- Background role within the same family
- Scene context and prop intensity
- Callout or label placement
- Product position

## Not Allowed

- New visual style per image
- Different background family
- Repeating the same exact background or prop shadow across the full set
- Different lighting mood
- Different product rendering level
- Repeating the same visual task
- Repeating the same callout composition
- Repeating the same hero layout after Image 1
